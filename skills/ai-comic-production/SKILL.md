# AI漫剧制作模板（从0到1完整流程）

## 模板说明
本模板基于抖音博主"一人漫剧（TW/AIGC）"的教学视频整理，适用于新手小白从零开始制作AI漫剧。主要使用工具：豆包（字节跳动AI）、即梦（AI视频生成平台）、小云雀（AI角色生成工具）。

## 制作流程

### 第一步：剧本准备
1. **剧本格式要求**：
   - 使用标准剧本格式
   - 包含场景描述、角色对白、动作指示
   - 建议每集时长1-3分钟
   - 文件格式：.txt或.docx

2. **剧本内容结构**：
   ```
   场景1：[场景名称]
   时间：[白天/夜晚]
   地点：[具体地点]
   
   [角色A]：（动作描述）对白内容
   [角色B]：（动作描述）对白内容
   
   [旁白/内心独白]
   ```

### 第二步：视频风格设定
1. **选择视频风格**：
   - 动漫风格（推荐）
   - 写实风格
   - 像素风格
   - 水彩风格

2. **画面风格设定**：
   - 色调：暖色调/冷色调/中性
   - 光影：自然光/戏剧光/柔和光
   - 质感：高清/复古/电影感

3. **格式锁定**：
   - 抖音/快手：9:16竖屏（1080x1920）
   - B站/YouTube：16:9横屏（1920x1080）
   - 小红书：3:4竖屏（1080x1440）

### 第三步：角色形象生成（使用小云雀）
1. **角色设定表**：
   ```
   角色名称：[姓名]
   年龄：[年龄]
   性别：[男/女]
   发型：[长度/颜色/造型]
   瞳色：[颜色]
   服装：[风格/颜色/配饰]
   性格特征：[关键词]
   标志性动作：[描述]
   ```

2. **生成提示词模板**：
   ```
   [风格]风格，[性别]角色，[年龄]岁，
   [发型]发型，[瞳色]瞳孔，
   [服装描述]，[表情]表情，
   [背景]背景，[光影]光影，
   高清细节，8K分辨率
   ```

3. **场景生成提示词**：
   ```
   [场景类型]，[时间]，[天气]，
   [主要元素]，[氛围描述]，
   [光影效果]，[色调]，
   电影感构图，高清细节
   ```

### 第四步：分镜制作
1. **分镜表格模板**：
   | 镜号 | 景别 | 角度 | 运动 | 画面描述 | 对白/旁白 | 时长 | 音效/音乐 |
   |------|------|------|------|----------|-----------|------|-----------|
   | 1 | 远景 | 俯拍 | 固定 | 城市全景 | 旁白 | 3秒 | 环境音 |
   | 2 | 中景 | 平拍 | 推进 | 角色特写 | 对白 | 4秒 | 背景音乐 |

2. **景别参考**：
   - 远景：展示环境全貌
   - 全景：人物全身+环境
   - 中景：膝盖以上
   - 近景：胸部以上
   - 特写：面部/物体特写
   - 大特写：极端细节

3. **角度参考**：
   - 平拍：客观视角
   - 俯拍：表现渺小
   - 仰拍：表现伟大
   - 斜拍：表现不安

### 第五步：视频生成（使用即梦）
1. **上传素材**：
   - 上传角色形象图片
   - 上传场景图片
   - 上传分镜脚本

2. **视频参数设置**：
   - 分辨率：1080p或4K
   - 帧率：24fps（电影感）或30fps（流畅）
   - 时长：根据分镜设定
   - 风格：与之前设定一致

3. **分镜替换**：
   - 将分镜中的元素替换为生成的图片
   - 调整构图和比例
   - 设置转场效果

### 第六步：后期调整
1. **提示词优化**：
   - 检查生成效果
   - 调整提示词关键词
   - 确保符合平台规则

2. **平台规则检查**：
   - 抖音：避免敏感内容，时长1-5分钟
   - 快手：注重原创性，避免搬运
   - B站：适合长内容，可添加弹幕互动
   - 小红书：注重视觉美感，短小精悍

3. **导出设置**：
   - 格式：MP4（通用）或MOV（高质量）
   - 码率：根据平台要求调整
   - 音频：确保清晰无杂音

## 常见问题解决

### 1. 角色形象不一致
- **解决方案**：使用固定的角色提示词模板
- **建议**：保存每个角色的生成参数

### 2. 场景转换生硬
- **解决方案**：添加转场效果（淡入淡出、滑动等）
- **建议**：保持色调和光影一致

### 3. 视频生成失败
- **解决方案**：检查图片格式（推荐PNG）、大小（建议小于10MB）
- **建议**：分段生成，再合并

### 4. 提示词效果不佳
- **解决方案**：使用具体描述，避免抽象词汇
- **建议**：参考成功案例的提示词

## 进阶技巧

### 1. 角色一致性保持
- 使用固定的角色种子值（如果平台支持）
- 保存角色参考图，每次生成时上传
- 使用相似的提示词结构

### 2. 镜头语言优化
- 学习电影分镜技巧
- 使用动态镜头（推拉摇移）
- 注意节奏感和呼吸感

### 3. 声音设计
- 添加背景音乐（注意版权）
- 配音可以使用AI语音工具
- 音效增强沉浸感

### 4. 批量制作
- 建立角色库和场景库
- 模板化分镜脚本
- 批量生成提高效率

## 模板使用示例

### 示例1：制作一个3分钟爱情短剧
1. 准备剧本（300-500字）
2. 设定角色（男女主角+配角）
3. 生成角色形象（各5-10张不同表情）
4. 制作分镜（20-30个镜头）
5. 生成视频（分段生成，每段30秒）
6. 后期剪辑（添加音乐、字幕、转场）

### 示例2：制作一个系列漫剧（每集1分钟）
1. 建立角色库（5-10个固定角色）
2. 设计场景库（10-15个常用场景）
3. 批量制作分镜模板
4. 每周更新2-3集
5. 建立观众互动机制
