# 运镜库（50种镜头运动方式）

> 按5个技术模块分类，直接复制到提示词的"运镜规则"段

---

## 模块一：基础机位（10种）

| 编号 | 名称 | 英文 | 描述 |
|------|------|------|------|
| 01 | 固定机位 | Locked-off | 三脚架固定，画面完全静止 |
| 02 | 手持呼吸感 | Handheld breath | 手持拍摄，全程保持极其轻微的如呼吸般的镜头浮动 |
| 03 | 缓慢前推 | Slow push-in | 镜头缓慢向前推进，逐渐靠近主体 |
| 04 | 缓慢后拉 | Slow pull-out | 镜头缓慢向后拉开，逐渐远离主体 |
| 05 | 左右横摇 | Lateral pan | 镜头水平方向左右扫视 |
| 06 | 上下俯仰 | Tilt up/down | 镜头垂直方向上下扫视 |
| 07 | 跟随运动 | Follow tracking | 镜头跟随主体移动方向同步移动 |
| 08 | 环绕拍摄 | Orbit | 镜头围绕主体做圆周运动（顺时针/逆时针） |
| 09 | 升降镜头 | Crane/Jib | 镜头从低到高或从高到低垂直移动 |
| 10 | FPV自由视角 | FPV free-tilting | 第一人称自由倾斜视角，像无人机穿越 |

## 💡 Mx-Shell 最爱的组合

```
# 万能组合（几乎所有作品都用）
手持拍摄——全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。

# 氛围渲染
镜头缓慢前推，渲染氛围。

# MV舞蹈段
跟拍运镜，根据画面自动构图，跳切分镜，转换场景。

# 突发惊吓
画外传来声音→镜头随之一同产生应激性的0.1秒微颤。

# 两角色对峙
过肩长焦镜头，浅景深，焦点对焦于远处角色。
```
