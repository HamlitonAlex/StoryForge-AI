# 摄影机型号对照表

> 不同风格对应的摄影机+镜头组合，直接复制到提示词的"视觉基调"段

---

## 📸 为什么写真实摄影机比写"电影感"有效？

AI训练数据里大量真实电影的画面元数据都和具体摄影机型号绑定。给AI一个"IMAX胶片摄影机"，它就有了一个非常具体的视觉风格锚点——比你说"电影感强烈"这种模糊词高维多了。

Mx-Shell 原话："你可以去模拟真实的摄影机去拍摄。我比较喜欢这款摄影机，这款是诺兰导演经常会用的，就是拍《星际穿越》的那位导演。"

## 🎬 风格→摄影机对照表

### 1. 史诗感 / 大场面
```
视觉基调：变形宽荧幕电影质感。模拟IMAX胶片摄影机，搭配Panavision C系列镜头（焦段35mm，光圈f4）。
```
**代表作**：诺兰《星际穿越》《奥本海默》
**适用**：宏大场景、末日战场、太空

---

### 2. 暗调赛博 / 写实硬核
```
视觉基调：索尼威尼斯电影机+佳能K-35系列镜头。手持拍摄，全程保持极其轻微的如呼吸般的镜头浮动。
```
**代表作**：《银翼杀手2049》《沙丘》
**适用**：赛博朋克、暗调科幻、动作战斗

---

### 3. 港片 / 武侠
```
视觉基调：柯达35mm复古胶片，跳过漂白胶片质感。高光处柔和光晕。丰富纹理。
```
**代表作**：邵氏武侠片、王家卫电影
**适用**：武侠、复古港风、东方美学

---

### 4. 商业人像
```
视觉基调：Canon EF 85mm f/1.2。
```
**适用**：人物特写、商业广告、人像摄影

---

### 5. 原子朋克 / 复古未来
```
视觉基调：变形宽荧幕电影质感。IMAX胶片摄影机+Panavision C系列镜头（添加动态模糊）。60年代复古科幻美学。
```
**代表作**：美剧《辐射》
**适用**：复古科幻、60年代美学、原子朋克

---

### 6. 重型机甲 / 工业史诗
```
视觉基调：模拟IMAX胶片摄影机+Panavision C系列镜头。帧率24，模拟动态模糊。好莱坞青橙色调，低饱和度。
```
**代表作**：《环太平洋》《变形金刚》
**适用**：机甲、机械变身、大规模对战

## 💡 Mx-Shell 的核心建议

> "你可以去模拟我的粉丝就应该了解了，模拟真实的摄影机去拍摄。"

**记住这个万能公式**：
```
模拟{{摄影机品牌}}{{摄影机型号}} + {{镜头品牌}}{{镜头系列}}（{{焦段}}mm，{{光圈}}f{{值}}）
```

**最安全的默认选择**：
```
模拟IMAX胶片摄影机+Panavision C系列镜头（焦段35mm，光圈f4）
```
这个组合几乎适用于所有风格，是Mx-Shell用得最多的。
