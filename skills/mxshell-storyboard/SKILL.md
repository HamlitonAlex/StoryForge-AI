# 多镜头叙事模板

> 适用于：剪辑式短片，一个事件拆分为N个镜头，每个5-8秒，然后组装成连贯叙事

---

## 骨架

```
[人物与场景设定]
  人物1（Portrait1参考）：……
  人物2（Portrait2参考）：……
  场景（Portrait3参考）：……
  声音：……
[氛围与画质]
  风格核心：……
  视觉基调：……
  色彩与影调：……
[故事内容]
  分镜一：景别+构图+运镜手法+画面内容
  分镜二：……
  分镜三：……
```

每个分镜必须包含四个字段：**景别 / 构图 / 运镜手法 / 画面内容**

---

## 五大高产技巧

### 1. 空镜 + 画外音
```
分镜：全景，平视拍摄吧台全貌。空镜，画面下方三分之一处为吧台。
画面内容：吧台后方传来悉悉簌簌的声音，像是有什么东西躲在吧台下翻找东西。
```
**用途**：没有主角+画外音，比任何可见怪物都更快建立"未知威胁"的紧张感

### 2. 过肩特写（建立两主体对视）
```
分镜：长焦，过肩镜头。
构图：机器人作为远景处于画面左侧，前景为鸵鸟后脑处于画面右侧。焦点对焦于机器人，产生浅景深。
画面内容：鸵鸟背对镜头迅速起身，后脑从画面下方入画窜出。机器人被吓到，双手抱头。
```
**用途**：两角色对峙时，OTS一个镜头捕捉两个主体——比来回切特写紧张感更强

### 3. 表情切换作为分镜终点
```
分镜：长焦特写，机器人侧面45度面部表情。
画面内容：机器人面部表情由黄色害怕切换至白色思考。
```
**用途**：一个分镜可以只讲一个情绪转变。表情切换=思考过程=内心独白

### 4. 跳切+跟拍+换场景（MV段落）
```
画面内容：机器人在场景各处（泳池边、二楼走廊、躺椅上、房顶）独舞。跟拍运镜，自动构图，跳切分镜，转换场景，穿插脚部特写。
```
**用途**：MV/蒙太奇不需要每帧精心设计。给AI跳切的权限，但锁定动作风格

### 5. 同期声列举
```
声音：不需要配乐，仅保留同期声。
```
有标志性环境音的场景要明确列举：玻璃破碎、脚步踩过碎屑、远处丧尸低吼、海浪声、风穿过破窗。

## 调试技巧

1. **先测2-3个分镜检查质感**，不要一上来就写7个
2. **分镜间的运动方向要衔接** — 上一个镜头主体向右移动，下一个镜头主体应从左侧入画
3. **每个分镜独立生成5-10秒**，后期剪辑拼接。30秒以上一镜到底的成功率极低
4. **Portrait编号必须稳定**：Portrait1永远是主角，Portrait2永远是配角——不要中途换

## 景别词汇速查

| 中文 | 英文 | 用途 |
|------|------|------|
| 大远景 | Extreme wide | 建立环境 |
| 远景 | Wide | 全身+大量上下文 |
| 全景 | Full shot | 完整人物+环境 |
| 中景 | Medium shot | 腰部以上 |
| 近景 | Close shot | 胸部以上 |
| 特写 | Close-up | 面部、手、道具 |
| 大特写 | Extreme close-up | 眼睛、嘴唇、纹理 |
| 长焦特写 | Long-lens close | 压缩景深 |
| 过肩 | Over-the-shoulder | 两角色对视 |
| 空镜 | Empty shot | 无人物，仅环境 |
