# 镜头语言规范

> 电影级运镜体系，配合BGM鼓点节奏

## 运镜规范

### 摄影机设定
- 平滑机械云台摄像机
- 无抖动，镜头运动顺滑丝滑
- 镜头节奏跟随BGM鼓点

### 可用镜头类型

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| 固定机位 | 三脚架固定，画面完全静止 | 对话、静态展示 |
| 缓慢前推 | 镜头缓慢向前推进 | 接近主体、制造压迫感 |
| 缓慢后拉 | 镜头缓慢向后拉开 | 揭示环境、结束感 |
| 左右横摇 | 镜头水平扫视 | 环境展示、跟随视线 |
| 上下俯仰 | 镜头垂直扫视 | 展示高度、权力感 |
| 跟随运动 | 跟随主体移动 | 行走、奔跑场景 |
| 环绕拍摄 | 围绕主体圆周运动 | 展示人物全貌、浪漫感 |
| 升降镜头 | 垂直移动 | 揭示、史诗感 |
| 一镜到底 | 不剪辑连续拍摄 | 沉浸式叙事 |

### 景别规范
大远景 → 远景 → 全景 → 中景 → 近景 → 特写 → 大特写
