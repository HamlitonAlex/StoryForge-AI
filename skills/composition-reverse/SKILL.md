# 构图反推 - 画面结构与时间线深度拆解

> 拥有20年好莱坞经验的专业电影导演、分镜师和AI视觉提示词专家方法论

## 使用方法
将视频内容描述、文案或截图提供给AI，本技能将严格按照"画面结构"和"时间线"进行深度拆解，生成可直接用于AI视频生成工具的高质量提示词。

## 拆解维度

### 一、画面结构拆解
1. **前景/中景/背景分层** - 识别画面中每个层的元素及其功能
2. **视觉焦点** - 确定画面主视觉焦点及其引导线
3. **构图法则** - 三分法、对称、引导线、框架构图等
4. **空间纵深** - 前景遮挡、透视收缩、大气透视
5. **色彩分区** - 画面色彩分布、冷暖对比、饱和度梯度

### 二、时间线拆解
1. **镜头分段** - 按运镜变化将视频拆分为独立镜头段
2. **运动轨迹** - 每段内主体运动方向、速度、加速度
3. **节奏标注** - 标注快/慢/停顿的节奏点
4. **转场方式** - 镜头段之间的过渡手法

### 三、提示词生成规范
生成的提示词必须包含以下段落结构：
- **[场景描述]** 环境空间、天气、时间
- **[人物状态]** 姿势、表情、动作
- **[镜头规则]** 运镜方式、景别、角度
- **[光影规则]** 布光方案、色彩倾向
- **[氛围定义]** 整体情绪基调、视觉风格
