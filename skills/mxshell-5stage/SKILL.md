## 关键技巧

### 1. 描述瑕疵 = 描述真实
```
❌ "完美的盔甲"
✅ "战损痕迹触目惊心，各处磨损掉漆，关节处有油污，表面有细微划痕"
```

### 2. 摄影机是质感锚点
```
❌ "电影感强烈"
✅ "模拟IMAX胶片摄影机+Panavision C系列镜头（焦段35mm，光圈f4）"
```

### 3. 呼吸感让画面活起来
```
✅ "手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感"
```

### 4. 反差感制造冲击
```
✅ "奢华度假别墅 ↔ 丧尸尸体"
✅ "机器人天真表情 ↔ 末日废墟"
✅ "MJ舞蹈 ↔ 战斗后死寂"
```

### 5. 同期声不要AI配
```
✅ "不需要配乐，仅保留同期声"
```
